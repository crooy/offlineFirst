Advantages of offline first

  • Reproducable build: not depending on IDE or computer
  • Saves a lot of time compared to manual build
  • Less human typing = less chance of mistakes
  • Runs your unit tests for you
  • Automatically deploy new builds - always up to date
  • Autmatically check if you did not break anything

Why Maven?

Maven is specifically known for dependency management

and project management options.

  • Uniformity: all Maven projects work the same way
  • Dependencies are managed for you
  • Document structure is forced / standardised
  • Provides project information such as versions etc
  • Maven Archetypes: project templates that others can use

Why not?

Maven is not very flexible.


The structure convention is forced.
